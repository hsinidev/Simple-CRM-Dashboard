## üöÄ Core Features

- **Modern & Responsive UI**: Built with Tailwind CSS for a beautiful experience on any device.
- **Full CRUD Functionality**: Easily Create, Read, Update, and Delete contacts.
- **Task Management**: Assign and track tasks for each contact to stay organized.
- **Local Data Persistence**: Your data is saved directly and securely in your browser's Local Storage. No cloud, no sign-up needed.
- **Data Portability**: Import and Export your contact list using Excel files (`.xlsx`).
- **Dynamic Filtering**: Quickly filter contacts by their status (Lead, Active, Complete).
- **SEO Optimized**: Includes a comprehensive SEO guide and rich JSON-LD schema for high visibility.
- **Stunning Visuals**: Features a beautiful, animated cosmic background for a premium feel.

## üèÅ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have [Node.js](https://nodejs.org/) (version 16 or newer) and a package manager like `npm` or `yarn` installed.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hsinidev/simple-crm-dashboard.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd simple-crm-dashboard
    ```

3.  **Install the dependencies:**
    ```bash
    npm install
    ```

4.  **Start the development server:**
    The command to start the development server will depend on the project setup (e.g., Create React App, Vite). A common command is:
    ```bash
    npm run start
    ```
    The application should now be running on [http://localhost:3000](http://localhost:3000).
